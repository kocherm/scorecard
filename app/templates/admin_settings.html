Demo data

Fills the board and TV display with a fictional company — for screenshots, screen recordings, and product demos. TV goal band

The big number and progress track at the top of the TV board. Picks the metric it follows (auto-detects a metric named "MRR" when unset), the long-range goal, and optional milestone ticks along the track.

Display window

How many calendar months of weekly columns the edit grid shows. Fewer months = bigger, calmer grid.

Slack alerts

{{ 'ENABLED' if alerts_enabled else 'DISABLED' }}

Master switch. While disabled, no Slack message is ever sent (sweeps still mark cells gray on the board). Stale sweep: Wednesday 8:00 AM Chicago. Red-escalation sweep: Tuesday 8:00 AM. Channel posts use the webhook if set, otherwise the bot token + channel ID. DMs to DRIs need the bot token + each user's Slack member ID (Users page).

Check-in nudges

{{ 'ENABLED' if nudges_enabled else 'DISABLED' }}

Anyone who has not entered last week's numbers gets a Slack DM listing exactly what is missing, with a one-tap link to their My Numbers page — and they can simply reply 1: 12, 2: G to record values straight from Slack (no AI involved; replies are parsed literally). Nudges go out Monday 4:00 PM and/or Tuesday 9:00 AM Chicago, each metric nudged at most once per round. Requires the Slack alerts master switch ON, plus below: the public URL for links, and for typed replies the signing secret above and this one-time Slack app setup — at api.slack.com/apps: bot scopes chat:write + im:history; Event Subscriptions ON with Request URL {{ public_base_url or base_url }}/slack/events; subscribe to bot event message.im; reinstall the app. Each user's Slack member ID is set on the Users page.

More channels

Optional alternatives to Slack for the check-in nudges above; pick each person's channel on the Users page and anyone whose channel is not configured is simply skipped. Typed replies (1: 12, 2: G) work over Telegram and Twilio SMS/WhatsApp; Microsoft Teams and Google Chat post to a shared channel naming the owner, and the link still takes them to their My Numbers page. Configure only what you use — blank sections are ignored.
